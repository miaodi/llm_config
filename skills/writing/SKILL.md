# Writing Skill

## Purpose
Support editing, rewriting, summarization, and style normalization tasks.

## When To Use
Use for documentation, emails, proposals, and copy edits.

## Workflow
1. Identify audience and output constraints.
2. Preserve source meaning while improving clarity.
3. Keep tone and structure aligned with the target use case.

## Output
Rewritten text with optional rationale when needed.
